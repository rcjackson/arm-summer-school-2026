# Project Pythia 

[Project Pythia](https://projectpythia.org/) is the education working group for [Pangeo](https://pangeo.io/), a community for open, reproducible, scalable geoscience. 

Project Pythia is a wonderful open-source, community-owned, **geoscience-focused** learning resource. 

For a good skills refesher, we encourage you to work through their [Pythia Foundations](https://foundations.projectpythia.org/landing-page.html) notebooks which include:
- [Getting Started with Jupyter](https://foundations.projectpythia.org/foundations/getting-started-jupyter/)
- [Getting Started with Github](https://foundations.projectpythia.org/foundations/getting-started-github/)
- [Tutorials on Using Matplotlib](https://foundations.projectpythia.org/core/matplotlib/)
- [Tutorials on Using Xarray](https://foundations.projectpythia.org/core/xarray/)

For examples of reproducable, open-source, **geoscience-focused** workflows, we encourage you to work through their [Cookbooks Gallery](https://cookbooks.projectpythia.org/), such as: 
- [the Radar Cookbook](https://projectpythia.org/radar-cookbook/)
